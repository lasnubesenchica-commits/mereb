# Mereb Nova · Prototipo

Rediseño de la plataforma escolar Mereb, enfocado en la experiencia del acudiente.

**Cómo verlo:** abrir `index.html` en cualquier navegador (no requiere servidor ni instalación).

- `index.html` — estructura de la aplicación
- `styles.css` — sistema de diseño (Fraunces + Nunito, materias con código de color)
- `data.js` — datos de demostración con la estructura real del sistema
- `app.js` — lógica de la interfaz (5 vistas: Semana, Notas, Mensajes, Pagos, Estudiante)

La vista Semana es imprimible directamente (Ctrl/Cmd+P).
